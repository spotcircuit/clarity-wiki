SpotCircuit is an agentic AI engineering practice. Six revenue streams, all built on the Clarity Framework.

## The 6 Revenue Streams

### 1. Clarity Framework
Open-source agentic intelligence framework. Community edition free, enterprise support paid.
- Self-learn loop, wiki system, 9 SE commands
- GitHub: https://github.com/spotcircuit/clarity-framework

### 2. AI Integration Consulting ($150-250/hr)
Wire AI into your existing stack. Config-driven pipelines, API connectors, workflow automation with n8n/Make/Zapier.

### 3. Claude Code Implementation
Custom Claude Code setups for engineering teams. Skills, slash commands, memory systems, MCP servers, hooks.

### 4. Knowledge Base Builder
The Karpathy wiki pattern as a service. Structured knowledge that compounds over time. Obsidian-compatible with auto-ingest.

### 5. Document Processing Pipelines
Multi-format ingest with hybrid AI. PDFs, transcripts, web clips, Slack threads into structured, searchable knowledge.

### 6. Build in Public
Open source contributions, technical blog posts, transparent engineering. Follow the journey.
