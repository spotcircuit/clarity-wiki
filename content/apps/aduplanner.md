# ADU Planner

#apps #aduplanner #real-estate #ai-vision #maps

AI-powered ADU (Accessory Dwelling Unit) planning. GPT-4 Vision analyzes satellite imagery for property type, buildable areas, setbacks. Interactive Google Maps drawing tools.

## Architecture

- **Frontend**: Next.js + TypeScript + Tailwind CSS
- **Maps**: Google Maps API (Places + JavaScript API)
- **AI**: OpenAI GPT-4 Vision (satellite imagery analysis)
- **Deployment**: Vercel

## AI Analysis Capabilities

Property type detection. Buildable area identification with size estimates. Setback measurements. Existing structure detection. Confidence scoring. Real-time ADU placement validation with compliance checking.
